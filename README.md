# SuperMario

SuperMario clone written in C++ using OpenGL 4.6 targetting mainly Linux-based operating systems.

## Build instructions

### Requisites

- premake5 v`^alpha16`
- opengl v`4.6`
- glfw v`^3.3.4`
- cxxopts v`^3.0.0`
- simdjson v`^1.0.2`
- gmake

#### Linux

On Debian based systems, install GLFW by using:

```shell
apt install libglfw3-dev
```
